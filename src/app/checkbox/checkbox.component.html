Checkboxes and Radio Buttons

Usually when we think of creating checkbox or radio buttons, we see them as the old school type where the checkbox is a silver white round bordered box and the radio button is somewhat similar except for the shape. Sometimes, in order to make them look better, we tend to use different UI Frameworks like Bootstrap or Foundation etc. In reality, that can be very easily done with simple CSS and an icons library as FontAwesome or Ionicons or any other of the kind.
